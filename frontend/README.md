# Restaurant Frontend UI

This is the frontend layer of a modular restaurant landing page system.

It is built with React and Tailwind-based styling structure and serves as the user-facing interface of the overall system.

---

## 🎯 Purpose

This frontend is designed to deliver a modern restaurant experience while remaining extensible for SaaS-based features such as:

- Digital menus
- Online ordering UI
- Reservation flows
- Admin dashboards (future integration)

---

## ⚙️ Tech Stack

- React (Create React App based structure)
- Tailwind CSS
- Modular component architecture

---

npm start
